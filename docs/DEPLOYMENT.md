# 배포 전략

## MSIX vs NSIS
- **MSIX**: 업데이트/무결성/스토어 친화적. 엔터프라이즈 배포 용이.
- **NSIS**: 유연한 설치 로직, 기존 Win32 친화적.

## 자동 업데이트
- MSIX App Installer 기반 자동 업데이트 또는 자체 업데이트 매니저.

## LibreOffice 포함 여부
- 포함 시 설치 크기 증가(수백 MB 가능), 초기 설치 편의성 상승.
- 미포함 시 최초 실행 시 탐색/설치 가이드 필요.

## Portable mode
- 설치 없이 실행 가능한 ZIP 모드 고려(권한 제한 환경 대응).

## 권장안
1. 초기 MVP: NSIS + LibreOffice 외부 의존(탐지/가이드)
2. 상용화: MSIX + 자동업데이트 + 선택적 번들 배포
